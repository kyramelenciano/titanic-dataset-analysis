# titanic-dataset-analysis
This repo contains my analysis of the Titanic dataset from Kaggle.

## src
This folder contains the jupyter notebook with the python code used to analyse the data.

## data
This folder contains the data set titanic.csv.

## result
This folder contains a pdf file with the report written of the findings from the analysis.

## doc
This folder is for additional documentation. I did not use any, so this is empty.
